金色力量黨主席童仲彥當過記者、主持人，也曾轉戰政壇任台北市議員，向來不避諱談情色議題，這回要下海拍A片當男優！ 童仲彥2018年因酒駕宣布退出議員選舉，去年初創立金色力量黨，致力推動「合法性產業專區」，還經營情趣用品線上商城。 《鏡週刊》報導，童仲彥表示，3年前議員退選後，有很多片商找他合作，當時沒答應，「一直到最近，覺得有很多理念想實現，才決定演出。」 「希望能夠拍出少見具教育意義的AV片」童仲彥指出，目前拍片模式和劇情仍與片商討論當中，「我想拍的不是只有接觸式的行為，希望用不同視角，呈現多元性平、性愛多元，還會有男男意向的劇情。」（即時新聞中心／綜合報導） 金色力量黨主席童仲彥當過記者、主持人，也曾轉戰政壇任台北市議員，向來不避諱談情色議題，這回要下海拍A片當男優！ 童仲彥2018年因酒駕宣布退出議員選舉，去年初創立金色力量黨，致力推動「合法性產業專區」，還經營情趣用品線上商城。 《鏡週刊》報導，童仲彥表示，3年前議員退選後，有很多片商找他合作，當時沒答應，「一直到最近，覺得有很多理念想實現，才決定演出。」 「希望能夠拍出少見具教育意義的AV片」童仲彥指出，目前拍片模式和劇情仍與片商討論當中，「我想拍的不是只有接觸式的行為，希望用不同視角，呈現多元性平、性愛多元，還會有男男意向的劇情。」（即時新聞中心／綜合報導） 金色力量黨主席童仲彥當過記者、主持人，也曾轉戰政壇任台北市議員，向來不避諱談情色議題，這回要下海拍A片當男優！ 童仲彥2018年因酒駕宣布退出議員選舉，去年初創立金色力量黨，致力推動「合法性產業專區」，還經營情趣用品線上商城。 《鏡週刊》報導，童仲彥表示，3年前議員退選後，有很多片商找他合作，當時沒答應，「一直到最近，覺得有很多理念想實現，才決定演出。」 「希望能夠拍出少見具教育意義的AV片」童仲彥指出，目前拍片模式和劇情仍與片商討論當中，「我想拍的不是只有接觸式的行為，希望用不同視角，呈現多元性平、性愛多元，還會有男男意向的劇情。」（即時新聞中心／綜合報導）
